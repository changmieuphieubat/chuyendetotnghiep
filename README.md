# Chuyên Đề Tốt Nghiệp: Phân Tích Dữ Liệu Giao Dịch Open Banking

## Mô tả
Dự án phân tích và dự báo dữ liệu giao dịch chuyển tiền từ hệ thống Open Banking của ngân hàng.

## 5 Chủ Đề Chính
1. Dự báo số lượng giao dịch/ngày
2. Dự báo số tiền giao dịch/ngày
3. Phân tích tương quan giữa số lượng và số tiền
4. Phân tích ảnh hưởng của ngày lễ
5. Phát hiện bất thường
